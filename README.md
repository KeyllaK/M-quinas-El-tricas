# Máquinas Elétricas
 Este repositório contém códigos desenvolvidos no MATLAB para a geração de gráficos utilizados na análise de máquinas síncronas, com foco em:
Curva de Magnetização (Ensaio a Vazio – OCC)
Curva de Curto-Circuito (SCC)
Ciclo de Histerese Magnética
Diagramas fasoriais

📁 Estrutura dos Códigos

🔹 1. Curva de Magnetização (OCC)
Simula a relação entre a corrente de campo (If) e a tensão induzida (E), considerando o efeito de saturação do núcleo magnético.

📌 Características:
Região linear inicial
Região de saturação

🔹 2. Curva de Curto-Circuito (SCC)
Representa a relação entre a corrente de armadura (Ia) e a corrente de campo (If) durante o ensaio de curto-circuito.

📌 Características:
Relação linear
Reta passando pela origem
Sem saturação do núcleo

🔹 3. Ciclo de Histerese
Simula o comportamento magnético de um material ferromagnético, mostrando o laço de histerese (B × H).

📌 Características:
Magnetização e desmagnetização
Saturação
Magnetismo residual
Força coerciva
